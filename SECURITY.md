# Security Policy

Thank you for helping keep AMCCS secure.

## Supported Versions

The repository is currently pre-release. The `main` branch receives fixes on a rolling basis and represents the supported version.

## Reporting a Vulnerability

1. Email `security@logiscan.dev` with a detailed description of the issue and, if possible, steps to reproduce it.
2. Do **not** file public GitHub issues for security bugs.
3. You will receive an acknowledgement within 3 business days. We will keep you informed as we investigate and address the report.

## Preferred Disclosure

Once a fix is available, we will coordinate public disclosure together. Please allow us reasonable time to investigate before sharing details publicly.
